ChaseFlix

This web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal inforamtion, and create a list of their favorite movies.

Endpoints

Business Logic URL HTTP Method Endpoint Request Body Data Format Response Body Data Format
Returns a list of all movies /movies GET None None Data from chaseflix_movies_db.json of all movies
Returns data about a single movie by title /movies/:Title GET :Title None Data from chaseflix_movies_db.json of the specified movie
Returns data about all movies with the same genre /movies/genres/:Name GET :Genre None Data from chaseflix_movies_db.json of movies with genres matching the searched genre
Returns data about all movies with the same Director /movies/directors/:Director GET :Director None Data from chaseflix_movies_db.json of the specified movies with the director searched by the user
Returns data about all movies with the same Actor /movies/actors/:Actor GET :Actor None Data from chaseflix_movies_db.json of the specified movies featuring the specified actor
Returns data about specific genre by name /genres/:Genre GET :Name None Data from chaseflix_genres_db.json of the specified genre
Returns data about a director by name /directors/:Name GET :Name None Data from chaseflix_directors_db.json of the specified director
Returns a list of all users /users GET None None Data from chaseflix_users_db.json of all registered users
Allows a user to be created /users POST None New data to chaseflix_users_db.json of the new user, structured: {"username": "", "password": "", "email": "", "Birthday": "month-day-year"} Updated data from chaseflix_users_db.json of the new user, structured: {"username": "", "password": "", "email": "", "Birthday": "month-day-year"}
Allows users to update their information /users/:Username PUT :Username New data to chaseflix_users_db.json of the user's new information, strucured: {"Username": "", "Password": "", "Email": "", "Birthday": "month-day-year"} Updated Data from chaseflix_users_db.json object of the user's new information, structured: {"Username": "", "Password": "", "Email": "", "Birthday": "month-day-year"}
Allows users to add a movie to their favorites /users/:Username/movies/:MovieID POST :Username, :MovieID None Updated data to chaseflix_users_db.json of the user's FavoriteMovies array
Allows user to delete a movie from their favorites /users/:username/movies/:MovieID DELETE :username, :MovieID None Updated data to chaseflix_users_db.json of the user's FavoriteMovies array
Allows a user to be deleted /users/:Username DELETE :Username None Text confirming the user has been deleted
